Run the scraper yourself

Scraping is rate-limited and at your own risk. Respect the target server.

git clone https://github.com/Katnapo/cook-meal-scraper.git && cd cook-meal-scraper\python-app && pip install flask requests && python cook_viewer.py
  • Opens a local web viewer at http://localhost:8080
  • Click Run Scraper to pull fresh data from cookfood.net (~45s)
  • Requires Python 3.9+
